# Getting Started
* **THE C PROGRAMMING LANGUAGE** by Brian W. Kernighan and Dennis M. Ritchie.
  > The only way to learn a new programming language is by writing programs in it.

* The first program to say "Hello, World!"
    ```C
      #include <stdio.h>

      int main(void) {

          return 0;
      }
    ```
